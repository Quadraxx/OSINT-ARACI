# OSINT ARACI

Kapsamlı bir Açık Kaynak İstihbarat (OSINT) aracı.

## Hedef

Bu proje; sosyal medya, e-posta, telefon numarası, IP adresi, domain ve daha birçok kaynaktan veri toplamayı hedefleyen, her şeyi kapsayan bir OSINT aracı olmayı amaçlamaktadır.

## Başlangıç Aşaması

Proje henüz başlangıç aşamasındadır. Temel yapı ve modüller oluşturulmaktadır.

## Kullanılacak Teknolojiler

- Python
- GitHub (versiyon kontrolü)
- Çeşitli OSINT kütüphaneleri ve API'ler
